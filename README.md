# Nuxt-Nest (Full stack NodeJS framework)

all nuxt and nest commands are available.

use `nuxt:` prefix for nuxt commands. like:
- `npm run nuxt:start`
- `npm run nuxt:dev`
- `npm run nuxt:test`
- etc ...

use this commands to run both nest and nuxt:
- `npm run stack:dev`

###### happy fullStack developing !
